## Using Icons 

Include React Component in your project:

    import Icon from '@eigi/bluebird-icons';

| Property   | Description                    | Type       | Default |
|------------|--------------------------------|------------|---------|
| width      | The width of the `svg` element | string     | '1em'   |
| height     | The height of the `svg` element| string     | '1em'   |
| fill       | The color used for the `svg` element | string | 'currentColor'
| className  | The width of the `svg` element | string | -  |
| style      | The width of the `svg` element | CSSProperties | - |
