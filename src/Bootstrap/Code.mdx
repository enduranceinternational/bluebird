# Code

## Inline

<p>
  Wrap inline snippets of code with <code>&lt;code&gt;</code>.
</p>

<p>
  For example, <code>&lt;section&gt;</code> should be wrapped as inline.
</p>

## User input

<p>
  Use the <code>&lt;kbd&gt;</code> to indicate input that is typically entered via keyboard.
</p>

<p>
  To switch directories, type <kbd>cd</kbd> followed by the name of the directory.
  <br />
  To edit settings, press <kbd><kbd>ctrl</kbd> + <kbd>,</kbd></kbd>
</p>

## Basic block

<p>
  Use <code>&lt;pre&gt;</code> for multiple lines of code. Be sure to escape any angle brackets in the code for proper rendering.
</p>

<pre>
  Sample text here...
</pre>

## Variables

<p>
  For indicating variables use the <code>&lt;var&gt;</code> tag.
</p>
