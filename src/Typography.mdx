## Things to note
- Never end a header or label with a period.
- Only use a period (.) for sentences, paragraphs or copy.
- Always use sentence case.
- Avoid use of all caps wherever possible

### Bluehost uses Open Sans exclusively.
- Open Sans is the only font used in the Bluehost design system
- Used for all titles, copy, buttons, form elements and navigation
- Should be styled as light, regular or semi bold only
- Only use italic if absolutely necessary
