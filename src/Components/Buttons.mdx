## Usage

Primary

- Reserved for primary actions on a given screen. (Think: 'Done', 'Save', 'Next', 'Submit')
- There should only be one primary button on any screen/scenario.
- If no clear primary action exists, consider using a different button style
- Can be used in isolation or in conjunction with a Secondary, Tertiary, Link style or Icon & Link button.
