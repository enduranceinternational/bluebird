# Loaders

There are several types of loaders available for usage depending on your needs.

## Content Loader

When the user may be waiting for content to be filled into a card or larger content area, giving them an idea for the layout is good practice. In Bluebird, we achieve this effect with [React Content Loader](https://github.com/danilowoz/react-content-loader).

React Content Loader has an [easy-to-use online tool](https://danilowoz.github.io/create-react-content-loader/) that allows you to create custom loaders based on your mockup.

### Defaults

Be sure to use the default values when configuring your custom loader:

| Prop           | Value   |
| -------------- | ------- |
| speed          | 2       |
| primaryColor   | #f2f4f7 |
| secondaryColor | #e6e9ec |
