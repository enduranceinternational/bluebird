# Breadcrumbs ![Ready Badge](https://img.shields.io/badge/component-ready-green.svg#badge)

Breadcrumbs provide a consistent way for users to orient themselves within a hierarchy of pages. Users are able to quickly go back through parent level pages in order to return to a previous screen.

## General Guidelines

- All links in a set of breadcrumbs should be clickable
- Breadcrumbs are a secondary navigational element and should not replace primary navigation

Add `active` attribute to make a `Breadcrumb.Item` active.

Do not set both `active` and `href` attributes. `active` overrides `href` and `span` element is rendered instead of `a`.
