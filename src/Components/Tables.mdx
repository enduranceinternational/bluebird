# Table

## Basic Table ![Not Ready Badge](https://img.shields.io/badge/component-not%20ready-red.svg#badge)

Use the `striped`, `bordered`, `condensed` and `hover` props to customise the table.

These props default to `false`.

## Responsive Table ![Experimental Badge](https://img.shields.io/badge/component-experimental-blue.svg#badge)

Add `responsive` prop to make them scroll horizontally up to `small devices` (under 768px).
When viewing on anything larger than 768px wide, you will not see any difference in these tables.

<Playground>
