## Dropdown in Nav ![Not Ready Badge](https://img.shields.io/badge/component-not%20ready-red.svg#badge)

Add dropdowns using the `NavDropdown` component.

<Playground>
  {/* import { Nav, NavItem, NavDropdown, MenuItem } from '@eigi/bluebird-react'; */}
  <Nav bsStyle="tabs" activeKey="1" onSelect={() => {}}>
    <NavItem eventKey="1" href="">
      NavItem 1 content
    </NavItem>
    <NavDropdown eventKey="2" title="Dropdown" id="nav-dropdown">
      <MenuItem eventKey="2.1">Action</MenuItem>
      <MenuItem eventKey="2.2">Another action</MenuItem>
      <MenuItem eventKey="2.3">Something else here</MenuItem>
      <MenuItem divider />
      <MenuItem eventKey="2.4">Separated link</MenuItem>
    </NavDropdown>
  </Nav>
</Playground>

## Nav underline style ![Ready Badge](https://img.shields.io/badge/component-ready-green.svg#badge)

Underline styled tabs can be build with one of:

- Nav element with nav-tabs-underline class
- Router, li, and Navlink elements by using "nav nav-tabs-underline" on a parent and an li element containing a NavLink or Link element.

<Playground>
  {() => {
  {/* import { Nav, NavItem } from '@eigi/bluebird-react'; */}
    const [activeKey, setActiveKey] = useState('1');
    return (
      <Nav
        className="nav-tabs-underline"
        activeKey={activeKey}
        onSelect={e => setActiveKey(e)}
      >
        <NavItem eventKey="1">All</NavItem>
        <NavItem eventKey="2">eCommerce</NavItem>
        <NavItem eventKey="3">Blog</NavItem>
        <NavItem eventKey="4">Portfolio</NavItem>
        <NavItem eventKey="5">Entertainment</NavItem>
        <NavItem eventKey="6">Food &amp; Drink</NavItem>
        <NavItem eventKey="7" disabled>
          News
        </NavItem>
      </Nav>
    );
  }}
</Playground>
