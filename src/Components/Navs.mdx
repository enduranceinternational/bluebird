## Dropdown in Nav ![Not Ready Badge](https://img.shields.io/badge/component-not%20ready-red.svg#badge)

Add dropdowns using the `NavDropdown` component.

<Playground>
  <Nav bsStyle="tabs" activeKey="1" onSelect={() => {}}>
    <NavItem eventKey="1" href="">
      NavItem 1 content
    </NavItem>
    <NavDropdown eventKey="2" title="Dropdown" id="nav-dropdown">
      <MenuItem eventKey="2.1">Action</MenuItem>
      <MenuItem eventKey="2.2">Another action</MenuItem>
      <MenuItem eventKey="2.3">Something else here</MenuItem>
      <MenuItem divider />
      <MenuItem eventKey="2.4">Separated link</MenuItem>
    </NavDropdown>
  </Nav>
</Playground>

## Justified ![Not Ready Badge](https://img.shields.io/badge/component-not%20ready-red.svg#badge)

## Nav underline style ![Not Ready Badge](https://img.shields.io/badge/component-not%20ready-red.svg#badge)

Underline styled tabs can be build with Router, li, and Navlink elements by using "nav nav-tabs-underline" on a parent and an li element containing a NavLink or Link element.

<Playground>
  <Router>
    <div className="nav-slider-wrapper mb5">
      <ul className="nav nav-tabs-underline">
        <li>
          <NavLink
            activeClassName="active"
            to={`/oneplace`}
          >
            Tab 1
          </NavLink>
        </li>
        <li>
          <NavLink
            activeClassName="active"
            to={`/anotherplace`}
          >
            Tab 2
          </NavLink>
        </li>
        <li>
          <NavLink
            activeClassName="active"
            to={`/athirdplace`}
          >
            Tab 3
          </NavLink>
        </li>
      </ul>
    </div>
  </Router>
</Playground>
